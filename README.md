# Getting Started:

### Installing

1. Clone the repo:

```bash
git clone git@github.com:HenrikLeutscher/POR1-FED1-HenrikLeutscher/tree/main
```

2. Install the dependencies:

```
npm install
```

### Running

To run the app, run the following commands:

```bash
npm run start
```
